## 🔧 KEY FILES

### Configuration Files
- **package.json** - Dependencies and npm scripts
- **.env** - Environment variables (API URL, timeout)
- **.env.example** - Template for .env
- **.prettierrc** - Code formatting
- **.gitignore** - Git ignore rules

### React Components
- **src/App.js** - Main application layout
- **src/components/ProductListing.js** - Product list container
- **src/components/ProductCard.js** - Individual product card

### Services & Utilities
- **src/services/apiService.js** - API communication
- **src/utils/helpers.js** - Helper functions

### Styling
- **src/styles/App.css** - Global styles
- **src/styles/ProductCard.css** - Card styles
- **src/styles/ProductListing.css** - List styles
